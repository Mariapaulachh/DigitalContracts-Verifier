FUNCIONES DEL SISTEMA

Nuestro sistema de consola permite:

REGISTRAR

Registra contratos con cláusulas, partes y fecha en segundos.

BUSCAR

Busca contratos por fecha, ID o tipo, con soporte bilingüe.

ORDENAR

Visualiza los contratos cronológicamente con Merge Sort.

VISUALIZAR

Explora cada contrato registrado con todos sus detalles.

ESTRUCTURAS UTILIZADAS

Este sistema implementa múltiples estructuras de datos:

HashTable

Almacena contratos con búsqueda por ID en tiempo constante.

AVLTree

Ordena contratos cronológicamente para búsquedas por fecha.

MultiList

Clasifica contratos por tipo (préstamo, seguro, inversión).

Merge Sort

Algoritmo eficiente para mostrar resultados ordenados.
